class bootstrapping_olympics.library.nuisances.NormalizeMin¶

Bases: bootstrapping_olympics.interfaces.rep_nuisance.RepresentationNuisance

Normalizes the streamels into a [0,1] range by removing the minimum and linearly scaling.

Transforms:

z[:n] = (y - min(y)) /  max(y) - min(y) 
z[n]  = min(y) 
z[n+1]= max(y)

To make this invertible, the streamels must have range in [0,1].
