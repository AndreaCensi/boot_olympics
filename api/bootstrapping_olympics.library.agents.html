test_agent Module¶

A simple agent useful for testing.

class bootstrapping_olympics.library.agents.test_agent.TestAgent(cmd)[source]¶

Bases: bootstrapping_olympics.interfaces.agent.AgentInterface

A simple agent useful for testing.

It executes any command specified on the constructor. If the command is a string, then it is eval()uated.

You can use the variables:

  • “t” for time.
  • “y” for the observations.
