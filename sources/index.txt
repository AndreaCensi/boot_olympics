BootOlympics
=============

|BootOlympics| is a Python package useful for running bootstrapping simulations and experiments.

The documentation is being written; not much is ready yet. Check back soon!

* For an introduction/tutorial, see the Wiki_.
* For reference, see the `API docs`__.

	Installation
	------------

	Install |BootOlympics| using: ::

	    $ pip install BootOlympics
    
	.. raw:: html
	   :file: download.html

	TYou can download this project in either zip_ or tar_ formats, or 
	download using git: ::

	    $ git clone git://github.com:AndreaCensi/boot_olympicsS.git

	Install using: ::

	    $ python setup.py develop
	    $ nosetests -w src         # run the extensive test suite
